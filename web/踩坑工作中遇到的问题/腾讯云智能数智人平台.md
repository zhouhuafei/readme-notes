# 腾讯云智能数智人平台
* 数智人平台致力于提供从数智人生产到运营管理全流程的服务。
* 主要包含 3 大模块：`形象资产生成与管理`、`交互数智人对话管理`、`播报数智人视频生成与管理`。
* 帮助客户打造数智员工，助力企业提升服务效率及用户交互体验。
