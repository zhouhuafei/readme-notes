## 登录云服务器
* 默认用户名为：`ubuntu`。
* 使用`ubuntu`用户登录云服务器之后，再使用`sudo passwd root`命令开启`root`用户登录。

## 如何安装依赖？
* 运行命令时，如果没安装，则系统会提示你安装。
* 例如你运行`docker --version`命令，但是你没安装`docker`，此时系统会提示你安装`apt install docker.io`。

## 常用命令
* apt-get update
* apt install docker.io
* apt install docker-compose
* 查看哪些服务开机自启动了：systemctl list-unit-files --type=service --state=enabled
* 查看哪些服务开机自启动了：systemctl list-unit-files --type=service --state=enabled | grep -E "docker|ssh"
* 开启docker开机自启动：systemctl enable docker
* 关闭docker开机自启动：systemctl disable docker
* 查看docker的自启状态：systemctl is-enabled docker

## 设置docker镜像源
#### vim /etc/docker/daemon.json
```json
{
  "registry-mirrors": ["https://docker.xuanyuan.me"]
}
```
#### 重启docker
* systemctl daemon-reload
* systemctl restart docker
#### 查看是否设置成功了
* docker info | grep "Registry Mirrors" -A 2
