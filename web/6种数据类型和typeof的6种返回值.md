# es5
* 6种数据类型
    - Object
    - String
    - Number
    - Boolean
    - null
    - undefined
* 5种原始类型
    - String
    - Number
    - Boolean
    - null
    - undefined
* 6种typeof返回值
    - 'object'
    - 'string'
    - 'number'
    - 'boolean'
    - 'function'
    - 'undefined'

# es6
* 新增原始类型
    - Symbol
* 新增typeof返回值
    - 'symbol'
