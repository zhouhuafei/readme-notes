# 常见内联块元素
> MDN 把这几个元素归类到了内联元素中，但是在谷歌浏览器上他们的默认属性是display:inline-block。
* input
* textarea
* select
* button

# 块元素的特性
* 支持所有CSS样式。
* 继承父级宽度。
* 独占一行。
* 代码换行不被解析。

# 内联元素的特性
* 不支持宽高，不支持上下margin，不支持上下padding。
* 内容撑开宽度。
* 同属性标签同行排列。
* 代码换行被解析为一个空格。
* 标签间距大小取决于父级的font-size。
