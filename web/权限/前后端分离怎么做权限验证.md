# token
* 登录，后端会返回一个token。
* 后续请求带上这个token即可。但是不安全哟，因为别人拿到这个token，也就拿到了你的登录权限。
    - 因为跨主域cookie带不过去，所以绑定到cookie上只是方便前端后续拿token，带给后端是带不过去的。
    - token的有效时间应该要和后端确认，对应的cookie也要存对应的时长。
    - 接口应该有个验签权限，并不是任何人都可以请求。
