# css
* 支持rpx，rem(scss配合媒体查询实现起来更容易)，vw，px。
* rpx会自动被百度小程序转成vw。

# navigator
* 1、 url要么相对路径```../xxx/yyy```，要么绝对路径反斜杠开头```/pages/xxx/yyy```
* 2、 open-type为navigate时，在模拟器中，如果是首页和详情页互相跳转，则1个循环之后，跳转就无效了。微信小程序是4个循环之后跳转就无效。

# 样式重置
```
image {
    width: 100%;
    height: 100%;
}

input {
    font-family: inherit;
    display: flex;
    align-items: center;
    flex: 1;
}

button {
    line-height: 1;
    border: none;
    border-radius: 0;
    outline: none;
    padding: 0;
}

/* 按钮去掉默认的边框 */
button::after {
    border: none;
}
```
