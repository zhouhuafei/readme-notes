```
var formData = new FormData();
formData.append('file', $('#file')[0].files[0]);
$.ajax({
    url: '/upload',
    type: 'POST',
    dataType: 'json',
    data: formData,
    cache: false,
    processData: false,
    contentType: false,
});
```
* cache设置为false。
    - 上传文件不需要缓存。
* processData设置为false。
    - 上传文件不需要对数据做处理。
* contentType设置为false。
    - 因使用了```FormData```对象，```Content-Type```默认就是```multipart/form-data```。
