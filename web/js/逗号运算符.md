# 使用逗号运算符可以在一条语句中执行多个操作
* 可用于声明多个变量：
```
var num=1, num=2, num=3;
```
* 逗号运算符总会返回表达式中的最后一项
```
var num = (5, 1, 3, 8, 0); // num 的值为 0
```
* 常用于for循环（逗号运算符允许将多个表达式视为单个表达式）
```
var i, j, k;
for(i=0, j=0; i<1, j<6; i++, j++){
    k = i + j;
}
// k 的值为 10
```

# 小技巧
* 交换变量，无需第三个变量：
```
var a = "a", b = "b";

//方法一
a = [b][b = a, 0];

//方法二
a = [b, b = a][0];
```
* 简化代码：
```
if(x){
    foo();
    return bar();
}else{
    return 1;
}

//可以简写成：
return x ? (foo(), bar()) : 1;
```
