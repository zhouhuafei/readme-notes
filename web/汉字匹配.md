* ASCII编码0-255之外的就是汉字
```
const re = /[^\x00-\xff]/;
re.test('汉字'); // true
```
* 16进制 00 的转换结果是
```
Math.pow(16, 0) * 0 + Math.pow(16, 1) * 0
```
* 16进制 ff 的转换结果是
```
Math.pow(16, 0) * 15 + Math.pow(16, 1) * 15
```

* 根据Unicode编码获取字符
```
String.fromCharCode(27721); // '汉'
```

* 根据位置获取字符的ASCII编码,如果超出了ASCII编码的界限则会获取Unicode编码
```
'汉'.charCodeAt(0); // 27721
```

* 根据位置获取字符
```
'汉'.charCodeAt(0); // '汉'
```
