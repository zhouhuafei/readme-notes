# windows安装

* 安装
```
choco install docker-for-windows --pre
```

* 安装目录
    - chocolatey的安装路径在哪，通过choco安装的东西就会在对应的地方。
    - 我本地把chocolatey的安装路径定制到了D盘
    - 详情请参阅同级目录里的chocolatey.md文件
    ```
    D:\chocolatey\lib\docker-for-windows\
    ```


* 更新
```
choco upgrade docker-for-windows --pre
```
