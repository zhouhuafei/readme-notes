## 松下商城小程序配置明细
* 正式环境
  - 公网带宽：100Mbps
  - 云服务器 - K8S之前：16C/64G x 2
  - 云服务器 - K8S之后：K8S多节点部署。服务节点自动扩缩容。
  - 主mysql云数据库：16C/64G x 1
  - 从mysql云数据库：8C/32G x 2
* 测试环境
  - 公网带宽：100Mbps
  - 云服务器 - K8S之前：8C/32G x 1
  - 云服务器 - K8S之后：K8S多节点部署。服务节点自动扩缩容。
  - 主mysql云数据库：8C/32G x 1
  - 从mysql云数据库：4C/16G x 2
* 压测机器：
  - 公网带宽：100Mbps
  - 云服务器：4C/16G x 3
